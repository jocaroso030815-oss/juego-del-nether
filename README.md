## Usar como extensión

Este repositorio puede ser añadido como una **extensión** en MakeCode.

* abra [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* haga clic en **New Project**
* haga clic en **Extensiones** en el menú del engranaje
* buscar **https://github.com/jocaroso030815-oss/juego-del-nether** e importar

## Edita este proyecto

Para editar este repositorio en MakeCode.

* abra [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* haga clic en **Import** luego haga clic en **Import URL**
* pegue **https://github.com/jocaroso030815-oss/juego-del-nether** y haga clic en importar
